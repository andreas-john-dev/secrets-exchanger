# secrets-exchanger — Project Memory

## Purpose
Secure, ephemeral secrets sharing web app. Users encrypt a secret message (+ optional passphrase), get a shareable link; the recipient decrypts it once and the secret is immediately deleted. All secrets auto-expire in 24h via DynamoDB TTL.

## Tech Stack
- **IaC**: AWS CDK v2, TypeScript, managed by **Projen** (`AwsCdkTypeScriptApp`)
- **Backend**: Node.js Lambda (NODEJS_22_X, ARM_64), API Gateway REST, DynamoDB, KMS
- **Frontend**: Angular 21, Angular Material 21, Tailwind CSS 3 (hosted in S3 + CloudFront)
- **Security scanning**: cdk-nag (`AwsSolutionsChecks`)
- **Testing**: Jest + CDK `Template` snapshot assertions
- **Bundling**: esbuild via `NodejsFunction`
- **Logging**: `@aws-lambda-powertools/logger`

## Key Constructs
- `StatefulStack`: DynamoDB `TableV2` (PK: `secretId`, TTL: `ttl`), KMS `SYMMETRIC_DEFAULT` key (alias: `SecretsExchangerKey`, rotation disabled)
- `SecretsExchangeLogic`: Two Lambda aliases (`Live`) — `StoreEncryptedSecretFunction` and `ReadSecretFunction`, each with 1024 MB memory, 1-week log retention
- `SecretsExchangerApi`: REST API with two resources: `POST /encrypt` and `POST /decrypt`; request model validation enforced
- `AngularWebApp`: Builds Angular app locally (`spawnSync`) and deploys to S3; CloudFront with geo-deny CN+RU, SPA error-page rewrites to `index.html`

## Encryption Flow (Double Encryption)
1. **Client → POST /encrypt**: body `{ secretString, passphrase? }`
   - Lambda: AES-256-CBC encrypt the JSON body → store only `encryptedData` in DynamoDB (NOT key/IV)
   - Lambda: KMS encrypt `{ secretId, key, iv }` with context `{ purpose: "secret-storage", application: "secrets-exchanger" }`
   - Returns `{ encryptedResponse }` (base64 KMS ciphertext) — this is the shareable token
2. **Client → POST /decrypt**: body `{ encryptedInput, passphrase? }`
   - Lambda: KMS decrypt `encryptedInput` → retrieve `{ secretId, key, iv }`
   - DynamoDB get by `secretId` → AES-256-CBC decrypt with key/iv
   - **Delete** the DynamoDB record immediately (burn-after-read)
   - Returns `{ secretString }`
